# Fridge

Fridge is a text editor.
**WORK IN PROGRESS.**

# TASKS

## done

* text editor (textarea-based)
* open / save text files
* markup for spaces and line breaks
* tab system for multiple documents
* automatic state backup
* document-wide search

## unfinished

* replace textarea with a fully customizable contenteditable editor
* support vertical writing mode
* load and represent text as AST document nodes for flexible paragraph rendering

* add features for series writing:
  1. scratchpad — a list-style worksheet for random ideas
  2. timeline — one or more chronological axes for storylines, each item linkable to AST nodes or groups
  3. wiki — a directory-based markdown wiki
(e.g. ../MyNovelSeries/wiki/characters/john.md)
